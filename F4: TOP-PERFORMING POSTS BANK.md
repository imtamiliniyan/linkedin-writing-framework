## Pattern Summary (for Claude to reference)

After reading all posts above, note these patterns:

**Hooks that consistently work for Tamil:**
- Confession + specific consequence ("I skipped X. It cost me Y.")
- Cost reveal with math ("$900 per client you are eating every single month")
- Tool announcement + immediate implication ("Apollo just did X. Here is what it means.")
- Paradox open ("Some agencies do X and get Y. Some do the same X and get Z. Why?")

**Body structures that drive saves:**
- Numbered steps with specific tool names
- Before state → what I built → after state with metrics
- Math breakdown → manual process description → automated alternative

**CTAs that drive DMs:**
- Comment [KEYWORD] and I will send you [specific thing]
- DM me [KEYWORD] and I will walk you through what I built

**CTAs that drive comments:**
- Binary question the ICP can answer in 3 seconds
- Status question ("Are you still doing this manually?")
